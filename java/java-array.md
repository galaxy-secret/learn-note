```
int[] a;

```

* tips

```
* 初始值
    创建一个数字数组时，所有元素都初始化为0。boolean数组的元素初始化为false。对象数组的元素初始化为null。
* 数组长度
    array.length 属性获得元素个数
    访问数组时，注意数组越界问题，会异常而终止。
* 长度在初始化时固定
    一旦创建了数组，就不能再改变它的大小（每个元素是可以改变的）。
    运行过程中需要改变扩展数组的大小，可以使用数组列表(array list)

```
